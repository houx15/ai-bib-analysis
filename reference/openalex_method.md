\section{AI-related scientific publications}

We constructed annual counts of AI‐related publications affiliated with China and the United States using the OpenAlex “works” bulk data. All compressed JSONL files in the works directory of the OpenAlex dump (snapshot updated in early 2025) were processed. For every work we extracted its OpenAlex identifier, title, publication year, and the list of authorships (including author positions and country information).
To identify AI‐related work, we relied on simple, transparent title matching. First, we discarded records with missing or empty titles. We then converted the remaining titles to lowercase and checked whether they contained at least one term from a curated list of AI‐related keywords (AI\_keywords in the code, for example terms such as “artificial intelligence”, “machine learning”, “deep learning”, or model names). A work was retained as “AI‐related” if any of these keywords appeared as a substring of its title; all other records were dropped at this stage.
Affiliation with a country was defined at the level of the work based on the country information recorded for its authors. For each AI‐related work, we first searched within the authorships field for a corresponding author (is\_corresponding == True) who had at least one associated country code in the countries field. If such a corresponding author existed, we took the first listed country code as the work’s country of affiliation. If no corresponding author with a country was available, we then searched for the first author (entries with author\_position == "first") with a non-empty countries field and again took the first listed country code. Works for which no author with a country code could be found were excluded. Each paper is assigned to only one country, to avoid duplicate counting. We then restricted attention to works whose inferred country code was in \{CN, HK, TW, US\}. In the analysis we interpret CN, HK, and TW as “China” (mainland China, Hong Kong, and Taiwan) and US as the United States.
For each retained work we recorded its OpenAlex ID, inferred country code, and publication year. These per-file records were written out as intermediate CSV files and then concatenated into a single table with one row per OpenAlex work. We then grouped this table by publication year and country code and computed the number of AI‐related works in each year–country cell, yielding yearly counts of AI‐related publications affiliated with China (CN/HK/TW) and with the United States (US).
